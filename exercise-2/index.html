Exercise 2

Using the page you created in Exercise 1, remove the script reference from the <head> and update your inline script tag to write the date and time directly to the page, using a common format (ex. 19:15, 11/5/2009).

These are some common date functions: 

		var date = new Date();  // creates a new Date variable
		date.getHours();        // the current hour of the day (24 hour format)
		date.getMinutes();      // the current minute of the hour
		date.getMonth();        // the number of the current month (ex. 11 for November)
		date.getDate();         // the current day of the month
		date.getFullYear();     // the year in YYYY format
